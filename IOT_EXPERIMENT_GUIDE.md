## 📱 WiFi 配置步骤

### 步骤 1: 获取 WiFi 信息
1. 打开手机或电脑的 WiFi 设置
2. 记下：
   - **SSID**（网络名称）
   - **Password**（网络密码）

### 步骤 2: 修改代码
对每个 `.ino` 文件，找到这两行并修改：

```cpp
const char* ssid = "YOUR_SSID";           // ← 改为你的WiFi名称
const char* password = "YOUR_PASSWORD";   // ← 改为你的WiFi密码
```

**示例：**
```cpp
const char* ssid = "ChinaNet-5G";
const char* password = "12345678";
```

### 步骤 3: 验证配置
在 Arduino IDE 中打开代码，检查：
- [ ] WiFi 名称已修改
- [ ] WiFi 密码已修改
- [ ] 没有多余的空格

## 💾 上传程序步骤

### 前置条件
- ✅ Arduino IDE 已安装
- ✅ ESP32 开发板库已安装
- ✅ USB 数据线已连接

### 上传流程

#### 1. 打开 Arduino IDE

#### 2. 选择开发板
```
Tools > Board > ESP32 Arduino > ESP32 Dev Module
```

#### 3. 选择串口
```
Tools > Port > COM? (ESP32)
```

#### 4. 打开代码文件
```
File > Open > [ex07/web_dimmer.ino]
```

#### 5. 检查代码
- 确认 WiFi 配置已修改
- 在代码中搜索 `YOUR_SSID`，确保没有遗留

#### 6. 上传
```
点击上传按钮（→ 符号）
等待"上传成功"提示
```

#### 7. 打开串口监视器
```
Tools > Serial Monitor
设置波特率: 115200
```

### 预期输出
```
Connecting to WiFi....
Connected. IP: 192.168.1.xxx
HTTP server started
```

## 🔧 常见问题与排除

### ❌ 无法连接 WiFi

**症状：** 串口显示 `Connecting to WiFi....` 但不连接

**排除步骤：**
1. 检查 WiFi 名称和密码是否正确
2. 尝试使用 2.4GHz 频率的 WiFi（某些 ESP32 不支持 5GHz）
3. 检查 WiFi 密码中是否有特殊字符
4. 重启 ESP32（按 Reset 按钮）

---

### ❌ 网页无法访问

**症状：** 浏览器显示"无法连接到服务器"

**排除步骤：**
1. 检查电脑/手机是否连接到同一 WiFi
2. 在浏览器中输入正确的 IP 地址
3. 检查防火墙设置
4. 尝试在串口监视器中 ping ESP32：
   ```powershell
   ping 192.168.1.xxx
   ```
5. 如果 ping 不通，重新上传程序

---

### ❌ LED 不亮或闪烁异常

**症状：** LED 没有亮或者闪烁不正常

**排除步骤：**
1. **硬件检查**
   - LED 极性是否正确（长脚为正极）
   - 电阻是否接入
   - 接线是否松动

2. **代码检查**
   - Ex07：拖动滑动条到最大，LED 应该最亮
   - Ex08：点击布防，手指触摸引脚 4，LED 应该快速闪烁
   - Ex09：不需要 LED

3. **引脚检查**
   - 确认代码中的 `ledPin = 2` 是否与硬件一致
   - 尝试在其他 PWM 引脚上测试

---

### ❌ 触摸引脚没有反应

**症状：** Ex08/Ex09 中触摸传感器无反应

**排除步骤：**
1. 检查引脚号是否为 4（ESP32 的触摸引脚）
2. 调整触摸阈值：
   ```cpp
   const int touchThreshold = 40;  // 减小值 = 更灵敏
   ```
3. 确保手指干燥且干净
4. 在串口监视器中监控原始数值：
   ```cpp
   Serial.println(touchRead(4));
   ```

---

### ❌ 网页刷新缓慢或卡顿

**症状：** 点击按钮或拖动滑动条有明显延迟

**排除步骤：**
1. 检查 WiFi 信号强度
2. 减少网络干扰（远离微波炉）
3. 确保路由器支持 2.4GHz
4. 重启 ESP32

## 📚 参考资源

### ESP32 触摸引脚
- 支持的引脚：GPIO0、GPIO2、GPIO4、GPIO12-GPIO15、GPIO27、GPIO32、GPIO33
- `touchRead()` 返回值范围：0-4095
- 值越小 = 手指越接近

### ESP32 PWM 引脚
- 所有 GPIO 引脚都支持 PWM
- 默认频率：5000Hz
- 分辨率：8 位（0-255）

### 网页通信
- **GET 请求**：`fetch('/setPWM?value=128')`
- **轮询间隔**：200ms（Ex09）
- **更新速率**：足够平滑，不会过度消耗资源

## 📞 故障排除清单

如遇到问题，按此顺序排除：

1. ✅ **硬件检查**
   - 连接是否正确
   - LED 极性是否正确
   - 电阻是否接入

2. ✅ **程序检查**
   - WiFi 名密码是否正确

   - 代码编译是否成功

3. ✅ **网络检查**
   - ESP32 是否成功连接 WiFi
   - 电脑/手机是否与 ESP32 同网
   - 防火墙是否阻止通信

4. ✅ **功能检查**
   - 各项功能是否按预期工作
   - 是否有延迟或异常
