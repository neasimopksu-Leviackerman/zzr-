# Ex03 - 使用 millis() 实现 SOS 闪烁信号

## 任务说明
使用 `millis()` 函数控制 LED 产生 SOS 闪烁信息。

## SOS 信号说明
SOS 是国际求救信号，在摩尔斯电码中表示为：
- **S** = · · · （三次短闪）
- **O** = — — — （三次长闪）
- **S** = · · · （三次短闪）
- 然后是一个较长的停顿后重复

## 时间参数
- 短闪（点）：100ms 亮 + 100ms 暗
- 长闪（划）：500ms 亮 + 100ms 暗
- 短闪和长闪之间间隔：100ms 暗
- 每个 SOS 播放完毕后停顿：1000ms 暗

## 硬件连接
- LED 正极：5V
- LED 负极：通过 220Ω 电阻连接到引脚 13

## 工作流程
1. 播放 SOS 信号（连续三次短闪、三次长闪、三次短闪）
2. 停顿 1 秒（LED 关闭）
3. 重复循环

## 测试方法
使用 Arduino IDE 的串口监视器（9600 波特率）查看信号状态和时序信息
