# Ex07 - Web 网页端无极调光器

## 任务说明
使用 ESP32 搭建网页服务器，实现通过浏览器滑动条控制 LED 亮度：
- 网页上显示一个滑动条
- 滑动条变化时通过 `fetch()` 发送数值到 ESP32
- ESP32 解析请求并将数值映射到 PWM 输出
- LED 亮度实时、平滑变化

## 硬件连接
- LED 正极 -> ESP32 引脚 2
- LED 负极 -> 220Ω 电阻 -> GND

## 使用说明
1. 修改 `web_dimmer.ino` 中的 `ssid` 和 `password`
2. 上传到 ESP32
3. 打开串口查看 IP 地址
4. 在浏览器中打开该 IP
5. 拖动滑动条，观察 LED 亮度变化
