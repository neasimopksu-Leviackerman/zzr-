# Ex08 - 物联网安防报警器模拟实验

## 任务说明
使用 ESP32 和网页控制实现简单安防主机：
- 网页提供“布防(Arm)”和“撤防(Disarm)”按钮
- 未布防时触摸引脚无反应
- 布防后触碰引脚，LED 开始高频闪烁并进入报警状态
- 只有网页点击“撤防”后才停止报警并恢复状态

## 硬件连接
- 触摸引脚 -> ESP32 触摸引脚 4
- LED 正极 -> ESP32 引脚 2
- LED 负极 -> 220Ω 电阻 -> GND

## 使用说明
1. 修改 `security_alarm.ino` 中的 WiFi 名称和密码
2. 上传程序到 ESP32
3. 打开串口获取 IP 地址
4. 使用浏览器访问页面，点击布防
5. 触摸传感器触发后，LED 开始闪烁
6. 点击撤防停止报警
